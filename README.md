# generator-astiapp

A generator for [Yeoman](http://yeoman.io).


## Getting Started

```
$ npm install -g yo
```

### Yeoman Generators

To install generator-astiapp from npm, run:

```
$ npm install -g generator-astiapp
```

Finally, initiate the generator:

```
$ mkdir something && cd $_
$ yo astiapp
```
